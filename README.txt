# 🎨 3D Portfolio Website

This is my experimental **3D portfolio project** where I explore the use of animations, creative layouts, and interactive design with **HTML, CSS, and JavaScript**.  
It’s not just a website — it’s a representation of my journey as a developer who loves to blend **design + code + creativity**.

---

## 🚀 Features
- 🎭 3D animated avatar integration  
- ✨ Smooth animations with JavaScript  
- 📱 Fully responsive design  
- 🎨 Dark theme with modern UI feel  
- 🖼️ Sections for About, Projects, and Contact  

---

## 💡 Purpose
I built this project to **challenge myself beyond basic websites** and experiment with animation libraries, avatar rendering, and scroll effects.  
It showcases my ability to:
- Learn new technologies  
- Build something visually unique  
- Tell my story through design  

---

## 🛠️ Tech Stack
- **HTML5** – structure  
- **CSS3** – styling and layout  
- **JavaScript (ES6+)** – animations and interactivity  

---

## 📖 What I Learned
- How to implement and customize **3D avatars**  
- Using **JavaScript for smooth scroll and animations**  
- Structuring a portfolio project for **real-world use**  
- The importance of **storytelling in portfolio design**
